Introduction to HTML

this is a guide to html please read through and try to remember what you have learned as you might need to reference it later.

What is HTML for anyway?

HTML is used to;

  • define the structure of a webpage
  • organize a webpage's content

Here is an example;

<p> this is for writing plain or paragraph text

One more;

<h1> This tag is for your primary heading, you should only have one and it should inform the reader and search engines of the primary subject matter of your page.
What about CSS?

CSS is short for cascading style sheets and is used for styling text and informing our browsers how our HTML code should be displayed.

Example;

.class will select a custom class you have defined for styling with CSS

Here is another example;

#id will choose all elements that you have given the specified ID to
What is CSS Used For?

CSS is useful for the following

  • Style our webpages
  • transform how elements work
  • optimize for devices

This code will set the font size of our headings to twice the root defined setting.

h1{font-size:2rem;}
Conclusion

Both HTML and CSS are needed to create a responsive, well designed website. HTML is easy to learn, CSS takes some time to understand. Rest assured it is well worth the time to learn!
